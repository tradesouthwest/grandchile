== Description ==
Custom style and scripts infuser for child theme of a parent theme without disturbing child theme.
Opens under Settings > GrandChild Editor 
Provides a low overhead editor to override child theme and parent theme styles.

== Features ==
* Select priority level of enqueue
* Saves to database so you can keep intact when deactivated.
* Enable or disable grandchild styles at any time.
* Named Grandchile incase there is someone who has plugin Grandchild.
* Also I was thinking of Jimi Hendrix's "Voodoo Chile" when I named it.

== Installation ==

This section describes how to install the plugin and get it working.
1. Upload `grandchile` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
